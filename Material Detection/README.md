# Computer_Vision
Computer vision projects using deep learning and object detection techniques including YOLOv8 for material classification.

A collection of computer vision projects built using deep learning frameworks and object detection models.

## Projects

### YOLOv8 Object Detection — Material Classification
Trained a YOLOv8 model to detect and classify product materials (plastic vs metal) from image and video inputs. Includes custom dataset training, model optimization, and real-time inference.

**Tech Stack:** Python, YOLOv8, OpenCV, TensorFlow
